On windows 10, install MongoDB with the following

1. install MongoDB from this site.
https://www.mongodb.com/download-center#community

2. click the .msi file

3. type mongod.exe from the bin folder

4. this will not work now, you need to create a /data/db folder in your c: drive

5. now in one command window type: mongod.exe

6. in another window type: mongo.exe

7. add the bin location to your PATH environment variable

8. test your database connection with opening a command window. type 
db

it will show "test"
This means the database is working.

9. Now its time to install Studio3T from the following site:
https://studio3t.com/
